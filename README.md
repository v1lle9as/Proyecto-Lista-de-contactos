Gestor de Contactos con Estructuras de Datos
Descripción

Este proyecto es una aplicación de consola en C++ que permite gestionar una agenda de contactos.
El sistema utiliza vectores, pila, cola y árbol binario de búsqueda para almacenar y organizar la información de los contactos (nombre, teléfono y correo electrónico).
